# simple-database
قاعدة بيانات تجريبية لاستخدامها مع Codex.

## مثال واجهة فلاتر

يوجد داخل المجلد `flutter_app` مثال بسيط لواجهة مستخدم بلغة فلاتر. يمكن تشغيله بواسطة الأمر:

```bash
flutter run
```

بعد الانتقال إلى ذلك المجلد:

```bash
cd flutter_app
flutter run
```

ستظهر رسالة نصية بسيطة في منتصف الشاشة.
